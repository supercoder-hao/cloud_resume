Campus Restroom Rating Website - Bruin Baño

Written in HTML and CSS and connected via Javascript to a Firebase database, Bruin Baños is a web application designed to rank and review restrooms across UCLA’s campus. Students can rate their restroom experiences and submit detailed reviews for others to see. With Bruin Baños, users gain valuable insights, helping them make informed decisions on the best campus restrooms to use.
